# mono-theme
A monochromatic theme for vim/neovim

### Install it

Via `Plug`
```
Plug 'felipevolpone/mono-theme'
```

or manually:
```bash
mkdir -p ~/.vim/colors
touch ~/.vimrc

wget https://raw.githubusercontent.com/felipevolpone/mono-theme/master/colors/mono.vim -O ~/.vim/colors/mono.vim
echo "colorscheme mono" >> ~/.vimrc
```
